View your Pest Control site

Easiest way (Mac):

Double‑click START-SERVER-AND-OPEN-SITE.command in this folder. Terminal will open, the server will start, and your browser will open. Keep the Terminal window open.

After the server is running, open:

Open site → http://localhost:3006

Or run manually in Terminal:

cd "/Volumes/Extreme SSD/Pest Control site"
npm run dev
Then go to http://localhost:3006

If the .command file won’t open: right‑click → Open → Open.
